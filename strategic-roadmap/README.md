# strategic-roadmap

Generates audit-shaped, evidence-backed strategic roadmaps for one repository from user seed ideas, repo mining, old plans, branches/bookmarks, and architecture scans. Use it to decide what repo-grounded work is worth planning before creating plan banks or feature planning artifacts.

This is not the ordinary `roadmap` skill for standard product/release roadmaps. It is intentionally above PR-level planning and produces a strategic roadmap with generated opportunities, ranked Now/Next/Later sequencing, architecture/deepening candidates, rejected/not-now items, success criteria, autonomy boundaries, and durable system-upgrade opportunities. If multiple repos are in scope, create one roadmap per repo.
